2. 가져다 쓴다? 참여한다
3. 장점과 단점은 동전의 앞과 뒤 - 귀찮음과 잘 정돈된 것. 하우가 아니라 와이가 중요하다.
4. 기술과 개발자 사이의 경계 - 기술 선택은 별거 아닙니다. 개발 프로세스중의 어느 한 점일뿐
